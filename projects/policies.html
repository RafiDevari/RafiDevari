Kebijakan Privasi

untuk Hi-Tensi (paket: com.catty.hitensi) — Berlaku sejak Oktober 2025

Kebijakan privasi ini menjelaskan bagaimana aplikasi Hi-Tensi menangani informasi. Data Anda hanya disimpan di perangkat Anda dan tidak dikirimkan kepada kami.

Ringkasan Tidak ada akun, tidak ada server, tidak ada berbagi data pihak ketiga. Hi-Tensi menyimpan data Anda hanya di perangkat. Jika aplikasi dihapus atau datanya dibersihkan, seluruh informasi akan hilang dari perangkat Anda.

1) Fungsi aplikasi

Hi-Tensi membantu pasien mencatat data screening, mengisi daftar tekanan darah (“tensi”), dan membuat pengingat/minum obat.

2) Data yang ditangani (hanya di perangkat)

  • Profil: nama dan umur yang Anda masukkan saat registrasi lokal.
  • Catatan kesehatan: jawaban screening dan catatan tensi yang Anda simpan.
  • Pengingat obat: jadwal alarm/pengingat yang Anda buat.

Disimpan menggunakan Flutter Shared Preferences / penyimpanan lokal (hanya di perangkat).

3) Penggunaan data

Semua pemrosesan dilakukan di perangkat Anda untuk:

  • Menampilkan riwayat catatan screening dan tensi.
  • Menjadwalkan serta menampilkan notifikasi pengingat obat.
  • Menyimpan profil dasar untuk personalisasi di aplikasi.

4) Pengumpulan & berbagi data

Kategori Dikumpulkan? Dibagikan?
Informasi pribadi (nama, umur) Ya – hanya disimpan di perangkat Tidak
Data kesehatan (tensi, jawaban screening) Ya – disimpan lokal Tidak
Lokasi, kontak, media Tidak Tidak
Identitas iklan/analitik Tidak Tidak

5) SDK pihak ketiga

Tidak ada. Aplikasi ini tidak menggunakan AdMob, Firebase, atau SDK pihak ketiga lainnya.

6) Akun

Tidak ada akun online. Profil lokal (nama, umur) bersifat opsional dan hanya tersimpan di perangkat.

7) Izin aplikasi

  • Notifikasi: untuk mengirimkan pengingat obat.
  • Alarm: untuk menjadwalkan pengingat sesuai waktu yang dipilih.

8) Keamanan

Data Anda hanya berada di ruang penyimpanan privat aplikasi di perangkat. Mohon jaga keamanan perangkat Anda (misalnya dengan kunci layar).

9) Pilihan & penyimpanan data

  • Anda dapat mengedit/menghapus catatan di dalam aplikasi (jika tersedia).
  • Anda dapat menghapus seluruh data aplikasi melalui pengaturan ponsel atau dengan mencopot aplikasi.

10) Anak-anak

Aplikasi ini tidak ditujukan untuk anak di bawah 13 tahun. Orang tua/wali dapat menghapus data aplikasi langsung dari perangkat.
